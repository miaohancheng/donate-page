使用方法：

  1. 确保页面引入了 jQuery。
  2. 在 HTML 中添加占位符 <div id="donate-widget-container" [data-attributes...]></div>。
  3. 在页面底部引入脚本 <script src="YOUR_PATH/donate-embed.js" defer></script>。
  4. 将脚本和图片资源托管在你的服务器上，并正确配置 URL。

详细说明请参考 项目 README 文件。
