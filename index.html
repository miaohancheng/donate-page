升级重点

保留旧接口，不阻断既有页面；新页面可直接采用 V2 能力。

  • 旧属性可直接沿用：`data-paypal-url` / `data-alipay-qr` / `data-wechat-qr`
  • 新增：`data-theme`、`data-lang`、`data-title`、`data-show-github`
  • 新增 API：`window.DonateWidget.init(selectorOrElement, options)`
  • 默认外链安全：自动使用 `rel="noopener noreferrer"`

快速接入

推荐使用 `data-donate-widget`，并继续兼容旧写法。

<div data-donate-widget
  data-paypal-url="https://paypal.me/your-name"
  data-alipay-qr="https://your.cdn/alipay.jpg"
  data-wechat-qr="https://your.cdn/wechat.jpg"
  data-asset-base-url="https://your.cdn/donate-icons/"
  data-title="Support This Project"
  data-lang="auto"
  data-show-github="false"></div>

<script src="/sample1/donate-embed.js" defer></script>
// Dynamic mount
window.DonateWidget.init('.sidebar-donate', {
  payPalUrl: 'https://paypal.me/your-name',
  showGithub: false,
  title: 'Tip Jar'
});
